# **Hotel Booking Cancellation Analysis Dashboard**

## **Project Overview**
This project focuses on analyzing hotel booking cancellation patterns using data analytics and Power BI visualization techniques. The dashboard helps identify factors influencing booking cancellations and provides business recommendations to reduce cancellation rates.

---

## **Tools Used**
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- Power BI

---

## **Project Workflow**

### **1. Data Cleaning**
- Removed duplicate records
- Checked missing values
- Cleaned dataset for analysis

### **2. Exploratory Data Analysis (EDA)**
Analyzed:
- Booking cancellations
- Lead time
- ADR (Average Daily Rate)
- Customer types
- Deposit types
- Repeated guests
- Monthly booking trends

### **3. Dashboard Development**
Created an interactive Power BI dashboard with:
- KPI Cards
- Cancellation Distribution
- Average Lead Time Analysis
- Average ADR Analysis
- Customer Type Distribution
- Repeated Guest Analysis
- Deposit Type Analysis
- Monthly Booking Trends
- Interactive Filters

---

## **Key Insights**
- Higher lead time bookings show higher cancellation probability
- No-deposit bookings have the highest cancellation rates
- Repeated guests are less likely to cancel bookings
- Average ADR is higher for cancelled bookings

---

## **Recommendations**
- Encourage advance deposits to reduce cancellations
- Offer loyalty benefits to repeated guests
- Monitor high lead-time bookings carefully
- Optimize pricing strategies based on seasonal demand
- Add customer job details during booking for better cancellation prediction
